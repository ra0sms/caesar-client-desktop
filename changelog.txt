CAESAR Desktop - Changelog
==========================

v1.0.4 (2026-06-23)
-------------------
- Refactor: extracted SessionManager from MainWindow (SRP, reduced coupling)
- Refactor: merged duplicate _pulse_sources/_pulse_sinks into _pactl_list(kind)
- Refactor: merged duplicate _sounddevice_inputs/_sounddevice_outputs into _sounddevice_list()
- Refactor: all network ports and magic strings moved to network/constants.py
- Refactor: version is now read directly from version.txt (no more version.py generation)
- Fix: removed unused duplicate network/ping.py (had socket leak)
- Fix: CAT Bridge signal leak on repeated Connect (old handler now disconnected)
- Fix: replaced bare except: with except Exception: in main_window.py
- Fix: added threading.Lock to FootswitchThread for thread-safe port access
- Fix: replaced proc.kill() with proc.terminate() in AudioRX/AudioTX (graceful shutdown)
- Fix: socket.shutdown() before socket.close() in CatBridge.cleanup() now guarded
- Fix: added socket.settimeout() to PingServer to prevent thread hang
- Fix: ServerMonitor now reuses one socket instead of creating a new one per iteration
- Perf: reduced footswitch poll rate from 20ms to 50ms (less CPU usage)
- Quality: added type hints to all functions across the entire project
- Quality: added __init__.py to audio/, network/, gui/ packages
- Quality: pinned dependency versions in requirements.txt
